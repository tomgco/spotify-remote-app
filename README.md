# Spotify App

## Installation

 1. Sign up for a [developer account on Spotify](http://developer.spotify.com/en/spotify-apps-api/developer-signup/)
 2. Open Terminal, `mkdir ~/Spotify`
 3. `cd ~/Spotify`
 4. `git clone git@github.com:tomgco/spotify-remote-app.git`
 6. Download the [latest version of Spotify](http://spotify.com/download)
 7. Open Spotify and type "spotify:app:remote" in the search bar (restart Spotify completely in case it doesn't find the App at first)
